# on AT Command Return result

When you send AT command to Xtron Pro, it will return information.

```sig
AT.onATReturn(function (atResult) {
	
})
```
### ~ hint

#### AT

Read https://docs.espressif.com/projects/esp-at/en/latest/AT_Command_Set/Basic_AT_Commands.html.The document contains all AT communication commands.

### ~
