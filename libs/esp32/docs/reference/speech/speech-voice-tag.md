# set Voice Tag

You can choose the character to play the text, whether it is male or female

```sig
speech.setVoiceTag(VoiceTag.Female)
```

## Parameters

* **tag**: voice tag
> * `Female`: female
> * `Male`: male

### ~ hint

#### speech 

All speech functions need to be connected to the network before they can be used normally

### ~
