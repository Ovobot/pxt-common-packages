# read Time In Format

Get current format time string from Xtron Pro.

```sig
rtcModules.readTimeInFormat(TimeFormat.BASICFORMAT)
```

Xtron Pro can read the hour, minute, second, year, month, day, etc.

### ~ hint

#### Time format

* `yyyy`: 4-digit year, with range 0001 to 9999. such as 2021
* `MM`: 2-digit month. such as 07
* `dd`: 2-digit day of month. such as 05	
* `hh`: 2-digit hour of a 24-hour clock. such as 14 
* `mm`: 2-digit minutes. such as 02 	
* `ss`: 2-digit seconds. such as 59 		
* `w`: 1-digit weak. such as 1. 1~7:Monday~Sunday

### ~

## Returns

* a [string](/types/string) which read the time string of the corresponding time format.
