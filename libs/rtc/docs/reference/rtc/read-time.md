# read Time

Get current time from Xtron Pro.

```sig
rtcModules.readTime(TimeType.SECONDS)
```

Xtron Pro can read the hour, minute, second, year, month, day, etc.

### ~ hint

#### TimeType

There are 7 time types available to choose from. seconds,minute,hour,day of the week,date,month,year.

### ~

## Returns

* a [number](/types/number) which read the time of the corresponding time type.
